# Rabeya Khatun - Portfolio

A modern, responsive single-page portfolio website for MERN Stack Developer Rabeya Khatun, built with React.js, Tailwind CSS, Framer Motion, and GSAP.

## Features

- **Modern Design**: Clean, minimalist design with professional color scheme
- **Responsive**: Mobile-first approach, works perfectly on all devices
- **Smooth Animations**: Framer Motion for component animations
- **Smooth Scrolling**: GSAP and Lenis for buttery smooth scrolling experience
- **Interactive Elements**: Hover effects and micro-interactions
- **Performance Optimized**: Fast loading and optimized for performance

## Sections

1. **Hero Section**: Eye-catching introduction with photo placeholder and call-to-action buttons
2. **About Section**: Brief introduction with key statistics
3. **Skills Section**: Visual representation of technical skills and tools
4. **Projects Section**: Showcase of 3 featured projects with live demos and GitHub links
5. **Contact Footer**: Social media links and contact information

## Technologies Used

- **Frontend**: React.js, HTML5, CSS3
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion, GSAP
- **Smooth Scrolling**: Lenis
- **Icons**: Heroicons (via Tailwind)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd rabeya-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build for Production

```bash
npm run build
```

## Customization

### Personal Information
- Update personal details in each component
- Replace placeholder images with actual photos
- Update social media links in `Contact.js`

### Projects
- Modify the projects array in `Projects.js` with your actual projects
- Update project images, descriptions, and links

### Skills
- Update the skills array in `Skills.js` with your proficiency levels
- Modify tools and technologies list

### Colors
- Customize the color scheme in `tailwind.config.js`
- Primary colors can be adjusted in the theme configuration

## Deployment

This project can be deployed to various platforms:

- **Vercel**: Connect your GitHub repository for automatic deployments
- **Netlify**: Drag and drop the build folder or connect via Git
- **GitHub Pages**: Use the build folder for static hosting
